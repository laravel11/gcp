# GCP VM Setup

This repository contains scripts to create and set up a VM instance on Google Cloud Platform (GCP).

## Prerequisites

- Install the Google Cloud SDK: https://cloud.google.com/sdk/docs/install
- Authenticate with your GCP account: `gcloud auth login`
- Set your GCP project: `gcloud config set project [PROJECT_ID]`
